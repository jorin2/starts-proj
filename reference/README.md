# 참고 자료 폴더 / Reference Materials

이 폴더에는 웹페이지 제작에 참고할 원본 자료를 넣어 주세요. 이 안내 파일을 제외한 내용은 `.gitignore`에 의해 공개 GitHub 저장소에 업로드되지 않습니다.

Place source materials for the website in this folder. Everything except this guide is excluded from the public GitHub repository.

## 추천 분류 / Suggested folders

```text
reference/
├── brand/       # 로고, 색상, 브랜드 가이드 / Logo and brand guide
├── copy/        # 소개 문구, 연락처, 기존 홍보물 / Copy and contact details
├── program/     # 커리큘럼, 일정, 대상, 강사 정보 / Curriculum and schedule
├── media/       # 원본 사진과 영상 / Source photos and videos
└── examples/    # 참고하고 싶은 웹사이트 캡처·링크 / Design references
```

파일명은 내용을 알아보기 쉽게 작성해 주세요. 예: `program/2026_curriculum.pdf`, `brand/logo_primary.png`.

민감정보, 개인정보, 사용 권한이 확인되지 않은 자료는 넣지 않는 것을 권장합니다. 공개 가능한 최종 이미지와 영상만 `assets/` 폴더로 옮깁니다.

Use descriptive filenames. Avoid sensitive, personal, or unlicensed content. Move only finalized, publishable media into `assets/`.
